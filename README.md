# knit-cli
前端开发脚手架工具

## 安装
```
npm install knit-cli -g
```

## 使用
```
  Usage: knit <command> [options]


  Options:

    -V, --version  output the version number
    -h, --help     output usage information


  Commands:

    list   List all the templates
    init   Generate a new project from a template
```
